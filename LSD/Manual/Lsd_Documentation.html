Lsd Documentation


Learning the use of a computer lanaguage is always a difficult task, and the best way is always to practice with exercises and studying existing examples. Lsd is no exception. The following documentation is thought for users without any prior knowledge of programming.

  • Slides of the Lsd course. As a first piece of documentation read the slides for a course on Lsd.
  • Lsd Manual. This document is a long paper describing how to build a model and listing all the Lsd functions. It should be handy to have it printed while coding models. This document may be slightly outdated. The on-line Help pages are the correct ones in case of differences. Open the Lsd Manual.
  • Examples. The distribution includes several example models. A relatively simple model containing most of the functions available in Lsd is the Social Diffusion model, located in the group Example Models/Literature. Understanding the code for the equations of this model will allow to implement almost any equation you have in mind.
  • Lsd Help pages. Each Lsd and LMM windows is endowed with a corresponding help page describing its use. The help pages be browsed to have a quick look at what Lsd can do. The initial point can be the table of contents of the Lsd Browser.
